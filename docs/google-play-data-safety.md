# Google Play Data Safety Notes

Last updated: April 1, 2026

Use this file as the basis for the Google Play Console Data Safety form.

## Data Types Used by the App

- Personal info: email address
- Financial info: sales, payments, receivables, payables
- App activity / user-generated content: customers, suppliers, products, stock movements, collection reminders
- Device or app info: basic technical data required for auth, sync, and notifications

## Purpose of Collection

- Account creation and authentication
- Business record storage and synchronization
- App security and fraud prevention
- Notification delivery for collection reminders

## Data Sharing

- Data is not sold
- Data is shared only with infrastructure providers required to run the app

## Account Deletion

- Users can delete their account inside the app
- Related data is deleted or anonymized unless legal retention is required

- Review the exact Play Console wording before final submission
- Make sure the final answers match the actual production behavior
