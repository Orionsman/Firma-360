# App Store Privacy Notes

Last updated: April 1, 2026

Use this file as a working draft for App Store Connect privacy disclosures.

## Likely Data Categories

- Contact info: email address
- User content: company, customer, supplier, product, stock, sales, payment, and reminder records
- Identifiers / account data: authentication-related account records
- Diagnostics or basic technical data: only if actually collected in production

## Purpose Mapping

- App functionality
- Account management
- Security and fraud prevention
- Customer support
- Notifications for collection reminders

## Important Review Rule

Only declare categories that match the actual production behavior of the app and connected services.
